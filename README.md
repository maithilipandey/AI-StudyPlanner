# AI Study Planner 📚🤖

An intelligent web application that helps students generate personalized study plans based on subjects, available time, and goals.

## 🚀 Features
- Personalized study schedule generation
- Subject-wise breakdown
- Weekly guidance
- Progress tracking
- Interactive UI 

## 🛠️ Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- PNPM
-
